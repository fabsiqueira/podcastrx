## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```
Or, you can also build a version: 

``` bash
yarn build
yarn start
```
Then, run the api server to get all the data : 

```bash
yarn server 
```


Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
